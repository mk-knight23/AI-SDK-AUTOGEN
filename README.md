# AI-SDK-AUTOGEN

[![AI-SDK Ecosystem](https://img.shields.io/badge/AI--SDK-ECOSYSTEM-part%20of-blue)](https://github.com/mk-knight23/AI-SDK-ECOSYSTEM)

> **Framework**: Microsoft AutoGen

Production-ready AutoGen SDK implementation with modern web stack.

## Tech Stack

- **Frontend**: Vue 3
- **Backend**: .NET 9
- **AI SDK**: Microsoft AutoGen

## Quick Start

```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend
cd backend
dotnet run
```

## Features

- AutoGen multi-agent framework
- RESTful API endpoints
- Docker containerization
- CI/CD pipeline
- Test coverage configured
